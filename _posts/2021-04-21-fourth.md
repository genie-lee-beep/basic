### 딕셔너리

중괄호{<키>:<값>,
       <키>:<값>,
       <키>:<값>
      }


```python
#값으로리스트의 요소 제거하기

list = [1,2,1,2]
list.remove(2)
print("remove(2):", list)

#리스트 내부에 해당되는 값이 여러 개 있어도 가장 먼저 발견되는 하나만 제거한다!
```

    remove(2): [1, 1, 2]
    


```python
#인덱스로 리스트의 요소 제거하기
#pop함수로 제거하기 
#<리스트>.pop(제거할 위치)

list = [0,1,7,8,4,5]
list.pop(3)
print("pop(3):",list)
print()
```

    pop(3): [0, 1, 7, 4, 5]
